# M09 — Observability

**Duration:** 5-6 days  

## Learning outcomes
- Instrument a service for the three signals
- Define & alert on SLOs
- Produce telemetry that AIOps consumes

## Analogy anchor (open with this)
Observability = a hospital's vital-signs monitor — not just alive/dead but trends, so you catch trouble before the flatline.

## Teach-the-trap
Alert noise. Teach symptom-based SLO alerts over cause-based spam.
