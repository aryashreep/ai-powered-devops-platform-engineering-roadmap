# M06 — Kubernetes (Core)

**Duration:** 8-10 days  
**Cert alignment:** CKA v1.35  

## Learning outcomes
- Deploy & operate a multi-service app on a real cluster
- Systematically troubleshoot (heaviest CKA domain)

## Analogy anchor (open with this)
Kubernetes = an orchestra conductor. Ensures 80 musicians play in sync; if one faints a backup steps in (self-healing).

## Teach-the-trap
Learners memorise YAML but can't debug. Weight labs the way the exam does — troubleshooting first.
