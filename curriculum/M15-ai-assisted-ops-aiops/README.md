# M15 — AI-Assisted Ops / AIOps

> ★ **differentiator**

**Duration:** 5-6 days  

## Learning outcomes
- Put AI to work as an assistant (human decides)
- Build a RAG-grounded incident assistant in shadow mode

## Analogy anchor (open with this)
AI-assisted ops = a very fast junior analyst who reads all the logs in seconds and hands you a summary — you still make the call.

## Teach-the-trap
Blind-apply of AI-generated IaC/manifests. Make 'human validates' a hard rule with a plausible-but-wrong suggestion lab.
