# M13 — Site Reliability Engineering (SRE)

**Duration:** 4-5 days  

## Learning outcomes
- Run reliability as an engineering discipline
- Execute an incident and a blameless postmortem

## Analogy anchor (open with this)
SRE = a hospital with service-level promises; when the error budget runs low, elective work pauses for emergencies.

## Teach-the-trap
Blameful postmortems. Grade on quality of action items, not blame.
