# M08 — GitOps

> ★ **differentiator**

**Duration:** 4-5 days  
**Cert alignment:** Argo CD / Flux  

## Learning outcomes
- Make Git the single source of truth for cluster state
- Implement progressive delivery with automated rollback

## Analogy anchor (open with this)
GitOps = a thermostat. Set desired state in Git; the controller constantly makes reality match and fights manual nudges.

## Teach-the-trap
Learners keep deploying via kubectl. Ban imperative deploys once GitOps is in.
