# 🗓️ Day 16 — HTTP & TLS

Today you'll learn the protocols that power the web — HTTP request/response and TLS encryption.

## Goal

Explain HTTP request/response flow, common methods and status codes, and how TLS provides transport security.

## Key Learnings

- HTTP/1.1, HTTP/2, HTTP/3 differences
- Methods: GET, POST, PUT, DELETE, PATCH
- Status codes: 1xx → 5xx
- Headers and request/response structure
- TLS handshake, certificates, and ciphers
- `curl` — the universal HTTP client

APIs, microservices, webhooks — everything speaks HTTP. Knowing status codes, headers, and TLS handshake is essential for debugging API integrations and securing traffic.
