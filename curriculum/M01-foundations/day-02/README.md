# 🗓️ Day 02 — Essential Linux Commands Toolkit

Welcome to Day 2! Today you'll build muscle memory for the commands you'll use every single day as a DevOps engineer.

## Goal

Navigate the filesystem, create and manipulate files, inspect system info, and use `man` pages to self-serve any command.

## Key Learnings

- Navigation: `cd`, `ls`, `pwd`, `tree`
- File operations: `mkdir`, `touch`, `cp`, `mv`, `rm`
- System info: `whoami`, `hostname`, `uname`, `uptime`
- Getting help: `man`, `--help`, `apropos`
