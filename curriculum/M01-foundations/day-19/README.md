# 🗓️ Day 18 — The One-Language Rule

Today you'll learn a career strategy: which languages to invest in for DevOps and why depth beats breadth.

## Goal

Choose the right language for the right job — Bash for glue, Python for automation, Go for performance-critical tooling.

## Key Learnings

- Bash: the universal glue language for system tasks
- Python: automation, API integration, AI/ML glue
- Go: infrastructure tooling, performance-critical paths
- When to use each — and when to switch

## Why This Matters

Jack of all trades, master of none is a trap. Focus deeply on one language while being fluent enough in others to read and debug. Most DevOps teams use Go (for tooling) + Python (for automation) + Bash (for shell tasks).
