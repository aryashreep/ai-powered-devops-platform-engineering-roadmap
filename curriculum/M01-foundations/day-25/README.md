# 🗓️ Day 24 — Signed Commits

Today you'll learn how to cryptographically sign your Git commits — proving they came from you and haven't been tampered with.

## Goal

Configure GPG or SSH signing for Git commits and tags, and understand why signed commits matter for supply chain security.

## Key Learnings

- GPG key generation and management
- Configuring Git to sign commits and tags
- Verifying signed commits
- Adding keys to GitHub/GitLab
- Supply chain security fundamentals

## Why This Matters

Signed commits are the foundation of software supply chain security. The "Verified" badge on GitHub proves the commit came from you, not an attacker. For open-source and enterprise repos, it's becoming mandatory.
