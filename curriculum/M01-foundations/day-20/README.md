# 🗓️ Day 19 — Bash for Ops

Today you'll learn to write production-level Bash scripts — the glue that holds DevOps workflows together.

## Goal

Write robust, maintainable Bash scripts that handle errors, accept arguments, and automate system tasks.

## Key Learnings

- Shebang, variables, and quoting
- Conditionals (`if`, `case`) and loops (`for`, `while`)
- Functions and argument parsing
- Exit codes and error handling (`set -e`, `trap`)
- Writing safe scripts (fail-fast, check dependencies)

## Why This Matters

Bash is the duct tape of DevOps. Entry-point scripts, CI steps, deploy scripts, cron jobs — most infrastructure automation starts in Bash. Writing them well prevents silent failures.
