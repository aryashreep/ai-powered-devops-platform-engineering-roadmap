# 🗓️ Day 23 — Git Hygiene

Today you'll learn the practices that make Git a pleasure to work with — for you and your team.

## Goal

Apply professional Git hygiene: conventional commits, .gitignore, commit hooks, and clean history management.

## Key Learnings

- Conventional Commits format (`feat:`, `fix:`, `chore:`, etc.)
- `.gitignore` — what to exclude and why
- Git hooks — automate checks on commit/push
- `git stash` — temporarily save work
- `git cherry-pick` — selective commit application
- Cleaning up stale branches

## Why This Matters

Good hygiene scales. Consistent commit messages enable changelog generation. Git hooks prevent secrets from leaking. Clean branches prevent merge hell.
