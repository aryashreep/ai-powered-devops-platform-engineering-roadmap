# M16 — Agentic Ops & AI Governance

> ★ **2026 frontier**

**Duration:** 6-8 days  
**Cert alignment:** emerging AI-security creds  

## Learning outcomes
- Move from AI-assisted to bounded-autonomous
- Design identity/guardrails/audit for safe agent action
- Red-team an agent

## Analogy anchor (open with this)
An AI agent = a capable new contractor. NHI = their badge; least privilege = key-card zones; audit log = sign-in sheet; circuit breaker = security escorting them out. MCP = USB-C for AI. Prompt injection via logs = a forged note in an inbox the agent was told to trust.

## Teach-the-trap
Broad permissions 'just to get it working.' The discipline is the opposite — start with almost nothing, widen on evidence.
